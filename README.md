## 📌 Project Summary
"Arcane Align" is an interactive 2D puzzle game developed using Java and the Swing GUI toolkit. The primary objective of the application is to test the user's short-term memory by challenging them to find matching pairs within a dynamically generated grid of face-down cards. 

This project successfully demonstrates the practical implementation of core Object-Oriented Programming (OOP) principles, including the encapsulation of game state, polymorphic event handling via ActionListeners, and UI component abstraction.

## 🚀 Features (Extra Stuff)
* **Dynamic Grid:** A fully randomized 4x5 layout every time the game restarts.
* **Custom Event Handling:** Custom `ActionListeners` capture mouse clicks for seamless gameplay.
* **Live Error Tracking:** Real-time UI updates to track the player's mistakes.
* **Non-Blocking UI:** Utilizes `javax.swing.Timer` to flip cards face down without freezing the game window.
